# 포트폴리오 사이트용 문구 - WeldVision

## 프로젝트명

WeldVision - 용접 결함 검출 및 해석 대시보드

## 한 줄 소개

용접 X-ray 이미지에서 결함 후보를 시각화하고, 결함의 형태적 특징을 바탕으로 위험도와 추정 원인을 제공하는 검사 보조 시스템입니다.

## 프로젝트 설명

WeldVision은 용접 결함을 단순히 분류하는 데서 끝나지 않고, 결함이 왜 위험한지와 어떤 원인으로 발생했을 가능성이 있는지를 함께 설명하는 품질 검사 보조 시스템입니다.

1단계에서는 C++ OpenCV를 사용해 결함 영역의 원형도, 종횡비, 밝기, 면적 등의 특징값을 추출하고 SVM 분류 실험을 진행했습니다. 2단계에서는 이 특징 분석 개념을 Python 기반 Gradio 대시보드로 확장하여, 전처리 결과와 위험도 스코어링, 원인 추론을 한 화면에서 확인할 수 있도록 구성했습니다.

현재 2단계는 YOLOv8 모델을 연결하기 전 단계입니다. 학습된 YOLOv8 모델이 없을 때는 OpenCV 기반 후보 검출 모드가 동작하며, 이는 최종 AI 판정이 아니라 전처리 결과에서 결함 후보를 확인하기 위한 보조 기능입니다.

## 주요 기능

- 용접 X-ray 이미지 업로드
- 원본 이미지와 후보 검출 결과 비교
- OpenCV 전처리 화면 제공: CLAHE, Black-hat, Gradient, Emboss
- 슬라이더 기반 실시간 재분석
- 특징값 출력: Circularity, Aspect Ratio, Mean Brightness
- 결함별 위험도 점수 제공
- 추정 원인 및 권장 조치 출력
- 한글 UI와 영어 기술명 병기

## 기술 스택

- C++17
- OpenCV
- SVM
- Python
- Gradio
- NumPy
- Pandas
- YOLOv8 예정

## 설계 포인트

이 프로젝트의 핵심은 1단계 결과를 버리지 않고 2단계의 해석 로직으로 재사용했다는 점입니다.

YOLOv8은 결함의 위치와 종류를 찾는 역할을 맡고, OpenCV 특징 분석은 결함의 형태적 근거를 설명하는 역할을 맡도록 설계했습니다. 이를 통해 단순 검출 결과가 아니라, 검사자가 이해할 수 있는 위험도와 원인 추론까지 제공하는 구조를 목표로 했습니다.
