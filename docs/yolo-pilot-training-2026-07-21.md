# YOLOv8 6클래스 파일럿 학습 결과 (2026-07-21)

## 목적

RIAWELC 폴리곤 JSON을 YOLO 바운딩 박스로 변환한 뒤, 데이터 변환부터 학습·검증·Gradio 추론까지 전체 경로가 실제 규모의 표본에서도 동작하는지 확인했다. 이 실험은 Mac 환경에서 수행한 6클래스 파일럿이며, 이후 Windows RTX 4070 환경의 RT 4클래스 모델과는 데이터 구성과 평가 범위가 다르다.

## 데이터 구성

원본 Training/Validation의 15개 폴더에서 폴더별 최대 100장을 고정 시드로 추출했다. 정상 이미지는 결함 클래스가 아닌 빈 라벨의 음성 샘플로 포함했다.

| 구분 | 이미지 | 음성 이미지 | 결함 객체 |
|---|---:|---:|---:|
| Train | 1,500 | 300 | 4,329 |
| Validation | 1,368 | 300 | 4,572 |

좌표가 비어 있거나 면적이 0인 객체는 Train 3개, Validation 4개를 건너뛰었다.

## 학습 설정

| 항목 | 값 |
|---|---|
| 모델 | YOLOv8n pretrained |
| Epochs | 10 |
| Image size | 512 |
| Batch size | 8 |
| Device | Apple M4 Pro MPS |
| Optimizer | AdamW (Ultralytics auto) |
| 소요 시간 | 약 8분 43초 |

## 전체 검증 결과

| Precision | Recall | mAP50 | mAP50-95 |
|---:|---:|---:|---:|
| 0.421 | 0.390 | 0.358 | 0.156 |

mAP50은 1 epoch의 0.056에서 10 epoch의 0.358로 상승했다. 마지막 epoch가 가장 높은 검증 성능을 기록했으므로 이 표본에서는 충분히 수렴하지 않은 상태로 판단했다.

## 클래스별 결과

| Class | Validation images | Instances | Precision | Recall | mAP50 | mAP50-95 |
|---|---:|---:|---:|---:|---:|---:|
| crack | 139 | 264 | 0.329 | 0.333 | 0.277 | 0.103 |
| porosity | 301 | 3,209 | 0.624 | 0.482 | 0.491 | 0.188 |
| lack_of_fusion | 233 | 332 | 0.380 | 0.170 | 0.192 | 0.083 |
| slag_inclusion | 202 | 409 | 0.337 | 0.186 | 0.188 | 0.065 |
| incomplete_penetration | 100 | 140 | 0.419 | 0.763 | 0.629 | 0.312 |

용입부족과 기공은 상대적으로 빠르게 학습됐고, 융합불량과 슬래그혼입은 Recall이 낮았다. 기공 객체 수가 다른 클래스보다 크게 많으므로 이후 실험에서는 이미지 수와 객체 수 불균형을 함께 확인해야 한다.

## Gradio 추론 확인

생성된 `best.pt`를 기존 추론 함수에 연결해 다음 샘플에서 검출을 확인했다.

- `VT_ST_03_14608597.jpg`: incomplete_penetration 0.776, 0.494
- `VT_ST_06_14627792.jpg`: undercut 0.309
- `VT_ST_02_14600480.jpg`: porosity 최고 0.606

일부 이미지에서는 검출 결과가 없었으므로 이 파일럿 모델을 최종 판정용으로 사용하면 안 된다. 모델 파일과 생성된 학습 결과는 Git에 포함하지 않는다.

## 후속 실험

2026-07-22에는 Windows RTX 4070에서 RT 4클래스 데이터로 별도 파일럿 학습을 진행했다. 해당 모델은 총 60 epoch 학습 후 mAP50 0.303을 기록했으며, RGB/BGR 입력과 중복 박스 처리를 수정해 Gradio 기본 모델로 연결했다. 두 실험은 클래스 수와 데이터 구성이 달라 수치를 직접 비교하지 않는다.

2026-07-24에는 클래스 균형을 조정한 RT v4 모델이 mAP50 0.413을 기록해 Gradio 기본 모델로 연결됐다. 다음 실험에서는 RT와 촬영 방식이 다른 VT 데이터를 분리하고, `incomplete_penetration`(용입부족)을 포함한 VT 전용 모델을 학습한다.
